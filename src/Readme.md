## Problem Statement: 
To Determine whether a one-to-one character mapping exists from one string to another string.
### Examples:
given s1 = abc and s2 = bcd, print "true" into stdout since we can map each
character in "abc" to a character in "bcd".
---------------------------------------------------------------------------------------------
### How to Compile
```commandline
javac Main.java
```
### How to Run
```commandline
java Main.java
```
